PROJECT 01
Enterprise GA4 Migration & BigQuery Integration
E-commerce · Super App
GA4 BigQuery GTM Data Layer

Led the full migration from Universal Analytics to GA4 for a large-scale super app platform ahead of the UA sunset deadline — covering web, iOS, and Android surfaces simultaneously. The scope included data layer architecture, GTM container restructuring, custom event taxonomy design, and BigQuery export configuration for downstream data science use.

The migration was delivered without any gap in KPI reporting continuity. Post-migration, the GA4 + BigQuery integration unlocked audience segmentation and remarketing capabilities that weren't possible in UA.

↑Zero KPI reporting downtime across the UA → GA4 transition
↑Advanced behaviour-based segmentation enabled for remarketing via GA4 + BigQuery
↑98%+ data collection accuracy validated post-implementation through structured QA playbooks
PROJECT 02
Predictive LTV Modelling & Retention Campaign Infrastructure
Fintech · CRM
CRM LTV Modelling Google Ads Hub Segmentation

Designed and delivered a unified data infrastructure combining performance marketing, CRM, and analytics data into a single model — enabling the business to predict customer lifetime value and build hyper-targeted retention campaigns at scale.

Integrated Google Ads Hub with the analytics stack to enable behaviour-based audience segmentation and remarketing. The LTV model informed campaign budget allocation across channels, shifting spend toward high-retention cohorts.

↑Unified CRM + analytics + paid media data into a single LTV prediction model
↑Hyper-targeted Drop Chase retention campaigns deployed at scale — improving ROI on retention spend
↑Behaviour-based remarketing via Google Ads Hub increased repeat engagement and LTV
PROJECT 03
Executive Reporting Automation & Self-Serve BI Platform
E-commerce · Leadership Reporting
Power BI SQL Looker Automation

Built the reporting infrastructure for an in-house analytics platform — translating business requirements into a self-serve BI layer tracking core KPIs (visits, transactions, conversion rate, error rates) across all business units. Co-built Power BI dashboards with BI engineering teams used in weekly leadership reviews.

A key outcome was reducing dependence on ad-hoc analyst requests — enabling product, marketing, and operations teams to answer their own questions from a single governed data source.

PROJECT 04
E-Commerce Funnel Analysis & CRO Programme
E-commerce · Conversion Optimisation
CRO Path Analysis A/B Testing UX Audit

Conducted deep behavioural and path analysis across the checkout funnel of a high-traffic e-commerce platform to identify where users were dropping off and why. Analysis covered session recordings, funnel visualisations, event-level data, and cohort comparisons across device types and acquisition channels.

Findings were translated into prioritised UX and product recommendations — tested via structured A/B experiments, with each iteration feeding the next hypothesis.

↑Identified key funnel drop-off points — reduced cart abandonment through targeted UX fixes
↑Structured A/B testing programme improved checkout conversion rate
↑Insights adopted by product and engineering teams — built into roadmap prioritisation
